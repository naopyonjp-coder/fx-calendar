FX収支カレンダー PWA セットアップ手順

【このアプリでできること】
・カレンダー形式で日別のFX収支を入力
・入力項目は「勝ち金額」「負け金額」のみ
・月別合計、年別合計を自動表示
・白ベースで明るいデザイン
・iPhoneのホーム画面にアイコン追加可能
・データは端末のブラウザ内に保存
・バックアップ/復元機能あり

【ファイル構成】
index.html        アプリ本体
styles.css        デザイン
app.js            動作ロジック
manifest.json     ホーム画面アイコン用設定
sw.js             オフライン起動用
icons/            アイコン画像

【iPhoneのホーム画面に表示する手順】
1. このフォルダ一式をWebサーバーにアップロードします。
   例：GitHub Pages、Netlify、Vercel、レンタルサーバーなど
2. iPhoneのSafariで公開URLを開きます。
3. 共有ボタンをタップします。
4. 「ホーム画面に追加」を選びます。
5. ホーム画面に「FX収支」のアイコンが追加されます。

【重要】
・このままiPhoneに直接ファイルを置くだけでは、通常のアプリのようなホーム画面運用はしづらいです。
・ホーム画面アイコン化するには、Web上に公開してSafariで開く方法が一番簡単です。
・データはブラウザのlocalStorageに保存されます。Safariのデータ削除、機種変更、別端末利用時はデータが消える可能性があります。
・機種変更前は、アプリ下部の「データ管理」→「バックアップ」を使ってください。

【カスタマイズしたい場合】
・アプリ名を変える：index.html の title / manifest.json の name, short_name
・色を変える：styles.css の :root 内の色設定
・アイコンを変える：icons フォルダ内のPNG画像を差し替え


【追加機能：ランダム格言】
カレンダー上部に投資・FX向けの短い格言をランダム表示します。
右側の「↻」ボタンを押すと、別の格言に切り替わります。
格言を増やしたい場合は app.js 内の TRADING_QUOTES に、
{ text: '格言本文', author: '表示名' } の形式で追加できます。


【著名人格言100件版】
カレンダー上部に、著名投資家・トレーダーの考え方を短く意訳した格言がランダム表示されます。
↻ボタンで切り替えできます。一巡するまで重複しにくいシャッフル方式です。
