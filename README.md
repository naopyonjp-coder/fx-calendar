# FX収支カレンダー

FXの収支をシンプルに記録するカレンダー形式のPWAアプリです。

## 公開URL

GitHub Pages:

https://naopyonjp-coder.github.io/fx-calendar/

## 主な機能

- 勝ち金額と負け金額だけを入力
- 日別収支をカレンダーに表示
- 月別合計と年別合計を表示
- データをブラウザ内に保存
- バックアップと復元に対応
- ホーム画面に追加してPWAとして利用可能
- 投資家・相場師の格言をシャッフル表示

## ファイル構成

- `index.html`: アプリ画面
- `styles.css`: デザイン
- `app.js`: カレンダー、保存、集計、バックアップ、格言表示
- `manifest.json`: PWA設定
- `sw.js`: オフライン用キャッシュ
- `icons/`: ホーム画面用アイコン

詳しいセットアップ手順は `README_セットアップ手順.txt` を参照してください。
